XTar - Resources

There are only two resources which are specific to XTar:

  • The helpDoc resource is a string which points to the pathname of the index.html file of the hypertext help pages.
  • The mappings resource is a list of file name endings to viewer mappings. This enables XTar to choose the correct viewer to run when the user requests to view a file.

Here is a copy of the resources file, XTar.ad, as supplied with XTar.

!!!! Help document !!!!
XTar*helpDoc:	/opt/xtar-1.5b/docs/index.html

!!! Some window dimensions !!!
XTar*TarWindow.width:	770

XTar*HelpForm.height:	640
