XTar 1.5b - X/Motif TAR File Viewer

XTar is a Motif program for viewing and manipulating Tape file ARchives. The main purpose of XTar is to allow the user to browse inside an, optionally compress, tar file without unpacking it.

The following topics are available to help you get to grips with the program:

  • Overview of the main window - extracting and viewing files.
  • The XTar pulldown menu - archive operations.
  • Set Attributes pulldown menu - extract options.
  • Help pulldown menu.
  • Statistics viewer - displays archive info.
  • Hypertext help navigator.
  • Internal file viewer.
  • XTar specific resources .
  • XTar as a CGI program - WWWTar .
  • XTar mailing list.
