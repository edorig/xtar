XTar - Mailing List

The XTar mailing list is for announcements only relating to XTar. If you would like to be informed of new releases and patches to XTar then you can subscribe to the list by sending an email message to: xtar-request@stella-consulting.ltd.uk with subject and body containing the word 'subscribe'.

You can also report any bugs in XTar to xtar-bugs@stella-consulting.ltd.uk.
